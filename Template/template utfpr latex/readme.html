[Federal University of Technology — Paraná] Academic Works

Go to PDF file of this LaTeX template


This LaTeX template (unofficial) allows the academic works (doctoral, master, and bachelor theses, etc.) production of the Federal University of Technology — Paraná (UTFPR).

It was developed based on the abnTeX2 academic works template, available at <http://www.abntex.net.br/>, which meets the standards requirements of the Brazilian Association of Technical Standards (ABNT) for development of technical and scientific Brazilian documents, as well as several code snippets developed by TeX-LaTeX Stack Exchange users, available at <http://tex.stackexchange.com/>.

Status: added by Luiz E. M. Lima, maintenance on demand.
Last updated: December 3, 2019 (Version 1.8).


Este modelo LaTeX (não oficial) permite a produção de trabalhos acadêmicos (teses, dissertações, trabalhos de conclusão, etc.) da Universidade Tecnológica Federal do Paraná (UTFPR).

Foi desenvolvido baseado no modelo de trabalhos acadêmicos do abnTeX2, disponível em <http://www.abntex.net.br/>, que atende aos requisitos das normas da Associação Brasileira de Normas Técnicas (ABNT) para produção de documentos técnicos e científicos brasileiros, bem como em diversos trechos de códigos desenvolvidos por usuários do TeX-LaTeX Stack Exchange, disponível em <http://tex.stackexchange.com/>.

Estado: adicionado por Luiz E. M. Lima, manutenção sob demanda.

  • ./makefile (Makefile para compilação no console do Linux).
  • ./readme.html (descrição e informações do modelo).

  • ./utfprpgtex.cls (classe de documento do modelo).
  • ./utfprpgtex.pdf (PDF resultante da compilação do modelo).

Os demais arquivos nos diretórios do projeto são imagens carregadas no modelo, em formatos suportados. Não apagar as imagens nos diretórios Ilustracoes.

2. Compilação dos arquivos

No diretório do projeto, abra um prompt de comando (no Windows) ou um console (no Linux) e digite make help para obter informações a respeito dos comandos de compilação dos arquivos do modelo LaTeX.

6. Dicas de uso

Sempre que for manipular e/ou substituir um dos arquivos constituintes do projeto, mantenha uma cópia do original num local seguro e/ou renomeie a cópia do original para que possa ser seguida como um exemplo no desenvolvimento do seu próprio arquivo.

O único formato de imagens que pode ser inserido ao compilar com LaTeX é o Encapsulated PostScript (EPS). Se for compilar com PDFLaTeX, para produzir um PDF diretamente, tem-se uma escolha mais ampla de formatos de imagens para inserir:

  • JPG, amplamente utilizado na Internet, em câmeras digitais, etc., sendo a melhor escolha para fotos.
  • PNG, um formato sem perdas muito comum (ainda que não tanto como JPG), sendo a melhor escolha para diagramas (se não for possível gerar uma versão vetorial) e capturas de tela.
  • PDF, é amplamente utilizado para documentos, mas pode ser usado para armazenar imagens também. Suporta imagens vetoriais e mapa de bits, mas não é recomendado para o último, pois JPG ou PNG irá fornecer o mesmo resultado com menor espaço em disco.
  • EPS, pode ser usado com ajuda do pacote epstopdf, que só precisa estar instalado. Portanto, não há necessidade de carregá-lo no documento.
